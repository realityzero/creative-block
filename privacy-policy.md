# Privacy Policy for Creative Block - Edit Text, Images & Colors on any page Extension

**Last Updated:** 25 Feb 2026

## Overview
Creative Block is a browser extension that allows users to temporarily edit webpage elements locally in their browser.

## Data Collection
This extension does NOT collect, store, or transmit any user data. All modifications are:
- Made locally in your browser only
- Temporary and session-based
- Never sent to external servers
- Not tracked or monitored

## Permissions Used
- **activeTab**: To access the current webpage you're editing
- **scripting**: To enable design mode and editing features
- **contextMenus**: To provide right-click menu options
- **<all_urls>**: To work on any website you choose to edit

## Third-Party Services
This extension does not use any third-party services or analytics.

## Changes to Privacy Policy
Any updates will be posted on this page and in the Chrome Web Store listing.
